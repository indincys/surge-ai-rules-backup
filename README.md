# surge-ai-rules-backup

Backup repository for Surge AI dependency rulesets.

## Quantumult X

- AI merged ruleset: https://raw.githubusercontent.com/indincys/surge-ai-rules-backup/main/QuantumultX/VoyagerAI.list
- AI local Voyager rules only: https://raw.githubusercontent.com/indincys/surge-ai-rules-backup/main/QuantumultX/VoyagerAI.local.list
- Voyager remote ruleset replacements: https://raw.githubusercontent.com/indincys/surge-ai-rules-backup/main/QuantumultX/VoyagerRemoteRulesets.qx.txt

The `QuantumultX/` directory contains Quantumult X filter-format conversions for the AI rules used by Voyager, plus generated Quantumult X replacements for Voyager's active remote rulesets.
